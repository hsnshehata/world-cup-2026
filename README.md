# World Cup 2026 - Interactive Globe

A beautiful interactive website for FIFA World Cup 2026 with focus on Egypt and Arab teams.

## Features
- 🌍 Interactive 3D Globe with stadium markers
- 🇪🇬 Egypt national team dedicated section
- 🌙 Arab teams section
- ⏱️ Live countdown to matches
- 📊 Match predictions
- 🏟️ Stadium details with match schedules
- 📱 Fully responsive (mobile + desktop)
- 🎨 Dark theme with gold accents

## Tech Stack
- Vanilla HTML/CSS/JS
- Three.js for 3D Globe
- Tailwind CSS (CDN)
- No build tools needed

## Deploy
```bash
docker build -t world-cup-2026 .
docker run -p 3000:3000 world-cup-2026
```
